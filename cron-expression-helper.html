Cron Expression Helper

Creador i explicador d'expressions cron

Constructor visual
Expressió cron
0 0 * * *
Explicació

S'executa cada dia a mitjanit (00:00).

Pròximes 5 execucions
Ajusta els valors per veure les pròximes execucions.
Consells sobre sintaxi cron
*
Qualsevol valor: coincideix amb tots els valors possibles
*/n
Cada n interval: p.ex. */5 cada 5 minuts
a-b
Rang: del valor a al b (p.ex. 1-5 = de dilluns a divendres)
a,b,c
Llista: valors separats per comes (p.ex. 0,30 = en punt i mitja)
@reboot
S'executa una vegada en iniciar el sistema
@daily
Equivalent a 0 0 * * * (cada dia a mitjanit)
@weekly
Equivalent a 0 0 * * 0 (cada diumenge a mitjanit)
@monthly
Equivalent a 0 0 1 * * (cada 1 del mes a mitjanit)
Referència crontab
Format:
min hour day mon dow
Camps:
minMinut (0-59)
hourHora (0-23)
dayDia del mes (1-31)
monMes (1-12)
dowDia setmana (0-7, 0=diu)
Operadors:
*Tots els valors
*/nCada n valors
a-bRang a-b
a,b,cLlista de valors
Exemples:
0 6 * * 1-5 Feiners 6AM
*/15 * * * * Cada 15 min
0 0 1 1 * Any nou
@reboot En iniciar
