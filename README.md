# Projectes ASIX

Col·lecció d'eines i utilitats web en HTML pur (sense frameworks ni dependències externes), desenvolupades per a pràctiques de cicle formatiu d'Administració de Sistemes Informàtics i Xarxes (ASIX).

Tots els projectes són autocontinguts en un sol fitxer HTML i funcionen directament al navegador.

---

## Xarxes i Sistemes

| Projecte | Descripció |
|----------|------------|
| **[network-topology-studio.html](network-topology-studio.html)** | Dissenyador de diagrames de xarxa drag-and-drop amb encaminadors, switches, PCs i núvols |
| **[subnet-calculator.html](subnet-calculator.html)** | Calcula xarxes, broadcasts, rangs d'hosts, màscares i classes IP amb visualització binària |
| **[vlan-calculator.html](vlan-calculator.html)** | Calcula rangs VLAN, màscares de subxarxa i capacitats d'amfitrions |
| **[port-scanner.html](port-scanner.html)** | Escàner de ports simulat amb base de dades de serveis comuns i anàlisi de resultats |
| **[port-reference.html](port-reference.html)** | Referència de ports de xarxa amb cerca, protocols i serveis associats |
| **[ip-converter.html](ip-converter.html)** | Conversor entre IP decimal, binari i hexadecimal |
| **[cron-expression-helper.html](cron-expression-helper.html)** | Crea, edita i explica expressions cron amb descripcions en llenguatge humà |
| **[http-request-monitor.html](http-request-monitor.html)** | Simulador de peticions HTTP amb builder, resposta, historial i desglossament temporal |

## Com utilitzar-los

1. Descarrega o clona el repositori.
2. Obre qualsevol fitxer `.html` al teu navegador preferit.
3. No cal instal·lar res, ni servidor web, ni dependències.

Totes les eines funcionen 100% al client, sense enviar dades a cap servidor.

---

## Requisits

- Un navegador web modern (Chrome, Firefox, Edge, Safari).
- JavaScript activat.
- Connexió a Internet **no** necessària (tret de funcions externes com la generació de QR).
